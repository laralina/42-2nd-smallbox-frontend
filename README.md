# 🎥 <i>small box</i>
## 스몰박스 소개 
<b>다양한 장르</b>의 영화를 상영하는 CGV 홈페이지를 모티브로,<br>
‘독립영화, 즉 <b>신선한 장르</b>와 <b>특별한 소재</b>를 좋아하거나 입문하고자 하는 관객에게 영화를 선보이자’는 취지로<br>
훌륭하지만 잘 알려지지 않은 독립영화를 소개하고 <b>누구든지 쉽게 접근할 수 있는 예매 서비스</b>를 제공합니다.<br>
